Help

                
                
Welcome to the Dot1xer Supreme Configurator! This tool generates advanced AAA configurations based on Wires & Wi-Fi templates (source).

                
How to Use

                
    
                    
  • Sidebar Navigation: Click the ☰ button to collapse/expand the sidebar. Select a step to proceed.
    • 
                    
  • Platform Selection: Choose your platform to proceed with configuration.
    • 
                    
  • Generate Config: Complete all steps and click to see the output.
    • 
                    
  • Download Config: Save the generated config as a text file.
    • 
                

                
Section Details and Examples

                
1. Platform Selection

                
Select the platform for your device. Example: Choose "Cisco: IOS-XE (Catalyst)" for a Catalyst 9300 switch.

                
2. AAA Settings

                
Configure the authentication method and accounting. Example: Select "RADIUS" and enable accounting for tracking user sessions.

                
3. RADIUS Servers

                
Add one or more RADIUS servers. Example: Add a server with IP "192.168.1.10", shared key "SecretKey", auth port 1812, acct port 1813, and priority 1.

                
4. TACACS+ Servers

                
Add one or more TACACS+ servers. Example: Add a server with IP "192.168.1.20", shared key "TacacsKey", auth port 49, and priority 1.

                
5. 802.1X Settings

                
Configure 802.1X settings for the selected platform. Example: For Cisco IOS-XE, set interface "GigabitEthernet0/1", VLAN 10, reauth period 3600s, TX period 10s, mode "Open", and enable IBNS 2.0 with policy "DOT1X_MAB". For Portnox, set profile "PORTNOX_PROFILE", guest VLAN 100, and enable dynamic VLAN.

                
6. CoA

                
Enable Change of Authorization (CoA). Example: Enable CoA and set port to 1700.

                
7. Device Tracking

                
Enable device tracking for IP tracking. Example: Enable device tracking with policy "IP-TRACKING".

                
8. Port Integration

                
Configure port settings. Example: Set port mode to "Access" and host mode to "Multi-Domain".

                
9. Generate Configuration

                
Generate and download the configuration. Example: After completing all steps, click "Generate Config" to see the output, then "Download Config" to save it as a .txt file.
